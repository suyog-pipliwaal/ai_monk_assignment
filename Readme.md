## Step to run
1. git clone https://github.com/suyog-pipliwaal/ai_monk_assignment.git
2. cd ai_monk_assignment
3. docker compose up --build
4. frontend runing at http://127.0.0.1:3000
5. backend runing at http://127.0.0.1:8000
